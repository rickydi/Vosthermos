# Site Web Vosthermos

Site web professionnel pour Vosthermos, spécialiste en réparation de portes et fenêtres.

## Formulaire de Contact WhatsApp Direct - SIMPLE POUR VOS CLIENTS

Le formulaire envoie les messages directement sur votre WhatsApp, sans que vos clients aient à utiliser WhatsApp.

### Comment ça fonctionne

1. **Pour vos clients (simple en un clic)**
   - Le visiteur remplit le formulaire et clique sur "Envoyer"
   - Le message est envoyé directement à votre WhatsApp
   - Le visiteur est redirigé vers la page de remerciement
   - Aucune autre action n'est requise de la part du visiteur

2. **Pour vous (notification WhatsApp)**
   - Les messages arrivent directement dans votre WhatsApp (+1 514 569 5583)
   - Format bien structuré avec toutes les informations du formulaire
   - Vous pouvez répondre immédiatement depuis votre WhatsApp
   - Notifications instantanées sur votre téléphone

### Configuration simple (à faire une seule fois)

1. **Activer le service CallMeBot** (Méthode par site web)
   - Allez sur https://www.callmebot.com/activate/free-api-whatsapp-messages/
   - Entrez votre numéro WhatsApp (+1 514 569 5583)
   - Utilisez votre adresse email pour créer un compte
   - Suivez les instructions reçues par email
   - Vous recevrez votre API key par WhatsApp et email
   - Remplacez "123456" dans le fichier js/email.js par votre API key

2. **Avantages**
   - Service GRATUIT pour un usage normal (quelques centaines de messages/mois)
   - Expérience utilisateur simplifiée - un simple clic sur "Envoyer" suffit
   - Messages reçus directement sur votre WhatsApp
   - Fonctionne parfaitement sur votre hébergement Lifelify

## Structure du Site

Le site est composé de 6 sections principales accessibles par défilement horizontal :

1. **Accueil** - Présentation générale de l'entreprise
2. **Quincaillerie** - Services de remplacement de quincaillerie
3. **Vitre Thermos** - Remplacement de vitres thermos
4. **Portes en Bois** - Réparation de portes en bois
5. **Moustiquaires** - Fabrication et réparation de moustiquaires
6. **Contact** - Formulaire de contact et informations

## Maintenance

Pour mettre à jour le contenu :

1. Textes : Modifiez directement les balises HTML dans `index.html`
2. Images : Remplacez les fichiers dans les dossiers correspondants sous `/images/`
3. Styles : Modifiez `css/styles.css` pour ajuster l'apparence
4. Fonctionnalités : Les scripts se trouvent dans le dossier `js/`
