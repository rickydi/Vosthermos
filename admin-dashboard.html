Configuration Google Analytics

  1. Va sur analytics.google.com
  2. Connecte-toi avec ton compte Google (Gmail)
  3. Cree une propriete pour vosthermos.com
  4. Copie ton ID de mesure (format: G-8NHVJ5P419)
  5. Remplace G-8NHVJ5P419 dans le code du site
  6. Tu verras tes stats en temps reel!

Configuration Search Console

  1. Va sur search.google.com/search-console
  2. Ajoute ta propriete vosthermos.com
  3. Verifie avec la methode "Balise HTML" ou "DNS"
  4. Soumets ton sitemap: https://www.vosthermos.com/sitemap.xml
